Endpoint API

GET /api/bali-calendar/today

Mengembalikan informasi kalender Bali untuk hari ini berdasarkan zona waktu WITA (UTC+8).

Contoh Permintaan:

curl https://kalender-bali-api.vercel.app/api/bali-calendar/today

Contoh Respons:

{
  "gregorianDate": "2025-07-10",
  "sakaYear": 1947,
  "sasih": "Kapat",
  "wuku": "Menail",
  "pancawara": "Kliwon",
  "saptawara": "Wraspati",
  "triwara": "Beteng",
  "lintang": "Pepe",
  "pararasan": "Laku Bulan",
  "ingkel": "Mina",
  "importantDays": ["Purnama Kapat"],
  "requiresLocalValidation": [],
  "isFullMoon": the,
  "isNewMoon": false
}

Kode Sampel:

curl https://kalender-bali-api.vercel.app/api/bali-calendar/today
fetch('https:k//alender-bali-api.vercel.app/api/bali-calendar/today')

Algoritma Perhitungan Wuku

Wuku dihitung berdasarkan siklus 30 minggu, dimulai dari referensi 1 Januari 2000 (Wuku Sinta). Algoritma:

graph TD A[Input Tanggal] --> B[Hitung selisih hari dari 1 Jan 2000] B --> C[Modulo 30 untuk indeks wuku] C --> D[Pilih wuku dari array WUKU] 
getWuku(date) {
    const days = this.getDaysDifference(date);
    const wukuIndex = (REFERENCE_WUKU + days) % 30;
    return WUKU[wukuIndex < 0 ? wukuIndex + 30 : wukuIndex];
}

Fase Bulan

Fase bulan untuk Purnama dan Tilem dihitung menggunakan astronomy-engine dengan threshold 0.98 untuk Purnama dan 0.02 untuk Tilem. 

isFullMoon(date) {
    const illum = astronomy.Illumination('Moon', date);
    return illum.phase_fraction >= 0.98;
}

Pertanyaan Umum

Apa zona waktu untuk /today?

Endpoint /api/bali-calendar/today menggunakan WITA (UTC+8).

Apakah hari baik selalu akurat?

Hari baik (Dewasa Ayu) memerlukan validasi lokal karena variasi antar daerah di Bali.

Bagaimana cara menangani error?

Periksa format parameter dan status respons (400 untuk input salah, 500 untuk error server).
